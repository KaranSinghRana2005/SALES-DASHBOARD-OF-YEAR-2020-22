# 📊 Sales Dashboard (2020–2022)

An interactive **Sales Dashboard** analyzing business performance from **2020 to 2022**, providing insights into sales, cost, quantity, and brand/category performance.

---

## 📊 Overview

This dashboard helps in understanding:
- Total Sales and Total Cost
- Total Quantity and Transactions
- State-wise sales distribution (map view)
- Brand-wise sales contribution
- Category-wise quantity analysis
- Supervisor-level performance

---

## 🚀 Features

- 🌍 Geographic analysis (Sales by State)
- 🏷️ Brand-wise performance breakdown
- 📦 Category-wise quantity insights
- 👨‍💼 Supervisor performance tracking
- 📊 KPI cards (Sales, Cost, Quantity, Transactions)
- 🎯 Clean and business-oriented UI

---

## 🛠️ Tools & Technologies Used

- Power BI /  Excel / SQL
- Data Cleaning & Transformation
- Data Visualization

---

## 📂 Dataset

- Dataset is **open-source / publicly available**
- Includes:
  - Sales data (2020–2022)
  - Product categories & brands
  - Transaction details
  - Regional (state-wise) data

## 📌 Key Insights

- Total Sales reached **99M**
- Total Cost recorded at **76M**
- Monitor and Mouse categories have the highest quantity sold
- Intel, Samsung, and Dell contribute major share in sales
- Sales are concentrated in major Indian states
- Significant number of transactions (~5095) recorded
